# Semana Omnistack v11

Desenvolvimento do Back End, Front End e Mobile da aplicação Be The Hero, desenvolvida na Semana Omnistack v11.

## Tecnologias utilizadas

### Back End
- NodeJS
- Express
- SQLite
- Knex
- Celebrate
- Jest

### Front End
- ReactJS
- Axios

### Mobile
- React Native
- Axios 
- Expo
